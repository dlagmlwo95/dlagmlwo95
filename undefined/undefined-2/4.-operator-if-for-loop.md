## 5. Equality

```javascript
// == != === !==
console.log(0 == false);     //true
console.log(0 === false);    //false
console.log('' == false);    //false
console.log('' === false);   //false
console.log(null == undefined);  //true
console.log(null === undefined); //false

```

## 6.Conditional operators

```javascript

  console.log('Welcome,ellie')
```

## 7. While

```javascript
let i =3;
while(i > 0){
  console.log(`while:${i}`)
  i--;
}
//while:3
//while:2
//while:1
```
