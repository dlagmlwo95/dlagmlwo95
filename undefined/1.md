## \#1-7 let, const, var

{% hint style="info" %}
const는 변수의 값을 바꿀수 없다.  
let은 변수의 값을 다른 값으로 바꿀  수 있다
{% endhint %}

```javascript
        let a = 221;
        let b = a - 5;
        a = 4
        console.log(b,a);  // 216, 4
```
