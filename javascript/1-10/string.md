# 문자 - string

## 6화 데이터타입 - 문자열과 숫자

```javascript
console.log('apple'.toUpperCase());                 //'APPLE'

'Hello world'.indexOf('o')                         // '4'

'         hello            '.trim()               // 'hello'

"1"+"1"                                          // '11'

// 검색법 : javascript string
```

{% hint style="info" %}
indexOf는 문자열이 몇번째에서 시작하는지 알려줌.   
 \* 숫자는 0부터 시작.
{% endhint %}
