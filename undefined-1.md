2-5화 불값\(boolean\)  
== : 같다\(값\)  
!= : 같지않다.  
=== : 같다\(자료형, 값\)  
!== : 같지않다.  
  
ex\)  '1' === 1;       -&gt; false  
       1 === true     -&gt; false  
       1 != '1'            -&gt; false  
       1 !== '1'           -&gt; true
