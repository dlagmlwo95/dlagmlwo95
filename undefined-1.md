# 제로초

1-3화  
\( \) : parentheses  
{ } : braces  
\[ \] : brackets

2-2화  
typeof로 문자열인지 확인 할 수 있다.  
ex\) typeof "asdf"     // typeof 0x1a1   // 

2-4화  
parseInt\('123'\) 문자를 숫자\(정수\)로 바꾸는 명령어

2-5화 불값\(boolean\)  
== : 같다\(값\)  
!= : 같지않다.  
=== : 같다\(자료형, 값\)  
!== : 같지않다.  
  
ex\)  '1' === 1;       -&gt; false  
       1 === true     -&gt; false  
       1 != '1'            -&gt; false  
       1 !== '1'           -&gt; true
