# 배열메소드1

push\(\) : 뒤에 삽입  
pop\(\) : 뒤에 삭제  
unshift\(\) : 앞에 삽입  
shift\(\) : 앞에 삭

### arr.splice\(n,m,x\) : 특정 요소 지움 

```javascript
let arr = [1,2,3,4,5];
arr.splice(1,2);
console.log(arr); // 1 4 5

let arr = [ 1, 2, 3, 4, 5];
arr.splice(1,0,100,200);
console.log(arr); // 1 100 200 2 3 4 5
```

### arr.splice\(\) : 삭제된 요소 반환

### arr.slice\(n,m\) : n부터 m까지 반

```javascript
let arr = [1,2,3,4,5];
```
